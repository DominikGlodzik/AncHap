# AncHap
Code for analysis of identity-by-descent in genetic data from isolated populations.

Packages required:
snpStats, devtools

open R session
library(devtools)
install('anchap')
library(anchap)

run demos on the toy data set
demo('toyData')
demo('chr2')

and run the attached example. Need to download a genetic map, see
details in the file.
source('runme.R')
